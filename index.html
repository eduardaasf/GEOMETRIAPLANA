Aplicação web educativa de geometria plana

Esta aplicação web foi desenvolvida para auxiliar os alunos do curso de matemática na disciplina de geometria plana.

A aplicação contém conteúdo sobre os principais conceitos de geometria plana, incluindo pontos, linhas, segmentos, ângulos, triângulos, quadriláteros, círculos e polígonos.

A aplicação também contém exercícios e atividades para ajudar os alunos a praticar os conceitos aprendidos.
